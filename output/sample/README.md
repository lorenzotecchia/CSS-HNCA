# Sample Output

Example output from a typical simulation run (100 steps, default config, seed=42).

## timeseries_example.csv

| Column | Description |
|--------|-------------|
| time_step | Simulation step (0-99) |
| firing_count | Number of neurons firing |
| avg_weight | Mean synaptic weight |

This sample demonstrates the expected format. Actual runs will have more steps and different values depending on configuration.
